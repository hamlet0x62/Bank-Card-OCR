# Bank-Card-OCR
A bank-card OCR system built with Flask, Vue and Tensorflow.
## 项目简介
该项目是一个基于深度学习的银行卡识别系统，是用于参加第九届`cnsoftbei`大赛的参赛作品。由于该届大赛已经结束，本项目也并未获得奖项，故现将其开源出来。若发现项目中存在值得改进的地方，还希望各位批评指正，共同进步。
